Esplora e gestisci i dati demo disponibili attraverso le API REST. Utilizza gli endpoint per creare, aggiornare o eliminare elementi di esempio.

API disponibili

  • GET /api/data – Elenca tutti gli elementi.
  • POST /api/data – Crea un nuovo elemento.
  • GET /api/data/<id> – Recupera un elemento specifico.
  • PUT /api/data/<id> – Aggiorna un elemento esistente.
  • DELETE /api/data/<id> – Rimuove un elemento.

Gli endpoint restituiscono risposte JSON e gestiscono automaticamente errori comuni come ID mancanti o payload non validi.
